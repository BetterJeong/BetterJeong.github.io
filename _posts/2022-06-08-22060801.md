## ✏️ 출력

입력  

```
system.out.println("출력할 내용");
```

출력  

```
출력할 내용
```

기본적으로 `system.out.println` 으로 출력할 수 있다.  
`println` 을 사용하면 출력 후 마지막에 엔터를 자동으로 쳐준다.  
엔터를 치고 싶지 않으면 `print` 로 사용하면 된다.  

인텔리제이에서는 `sout` 을 치고 엔터를 누르면 자동으로 `system.out.println` 을 쳐 준다.
